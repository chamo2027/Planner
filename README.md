# Planner – Online Event Planning System (SLIIT 2025)

## Overview
Planner is a web-based event planning system developed to simplify event creation, customization, and booking. The platform allows users to register, manage events, and customize event details easily.

## Features
- User registration and login
- Event creation and booking
- Event customization options
- Interactive homepage UI
- Basic database management for event data

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
